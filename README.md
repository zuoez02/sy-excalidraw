# SuperDraw
基于Excalidraw实现的挂件，基于现有excalidraw组件实现。

## 使用步骤

1. 插入挂件
2. 调整大小
3. 按照你的喜好添加内容吧


## 功能

### 界面功能

增加了右上角按钮支持保存和全屏

### 手写支持

基于Excalidraw实现手写功能支持，支持压感。目前在Matebook E上访问浏览器进行测试，未测试移动端、iPad等其他载体。

### 中文支持

已添加中文字体，使用面板中第四个按钮

### 自动保存

目前同时支持自动保存和手动保存
